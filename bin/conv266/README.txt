Usage: conv266
          -r root_dir            (full path to base dir)
          -c channel             (0 based camera number)
          -b begin_time          (YYYYMMDDhhmmss)
          -e end_time            (YYYYMMDDhhmmss)
          -f output_file_name    (full path with .mp4 or .avi
          -i input_file_name     (full path to input file to convert,
                                  -r -c -b -e will be ignored)
         [-k video_key]          (passwd for encrypted video)
         [-v bit_rate_in_kbs]    (default: 1500)
         [-o]                    (OSD on top of video, default: bottom)
